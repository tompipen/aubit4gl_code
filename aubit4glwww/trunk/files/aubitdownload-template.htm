Aubit 4gl download page

This server is on ADSL line (meaning it's slow), and is here primarily for nightly automatic binary builds; on top of that, it contains completely untested code, whatever is in CVS that night, will end up here.

Your preferred option should be to download RELEASED files from SourceForge Aubit4gl project page, and/or to get source code from Aubit CVS on SourceForge.

Available for download:

  • Source distribution
  • Precompiled binary distribution
  • Other files
  • Dependencies download links

Precompiled binary distribution

This is our first attempt at binary packaging, so please forgive us if it's not all it should be. Let me know if you have problems, but also please let me know in which configurations you found it to work OK. They are automatically built every night, so they should always be at most a day old. If they are not, let me know. We do not plan official releases until V 0.2

Filenames format:

  • Binary packages compressed archives: 
    aubit4gl-bin-[OS|kernel]-[iODBC/unixODBC/noODBC]-[GTK/noGTK]-[PDF/noPDF]-[version]-[build]-[build date].[tar.gz|rpm]
  • Compile platform info text files: 
    compile-platform-info-[full binary package name].txt

Platforms available:

  • Linux tar.gz and RPM precompiled binary packages
  • Microsoft Windows
  • Other operating systems

Please note:

  • No dependencies build DOES NOT include ODBC functionality, therefore you CAN NOT execute any database oriented (SQL) commands!
  • Typical size of tar.gz packages is between 2 and 3 MB

Dependencies download links

Depending on your choice of binary package, or options while compiling source distribution, you will need some or all third-party software and/or libraries listed here present on your system. First, check if you have them, and if they are compatible version. And if you don't, follow links provided, and install them according to there installation instructions. See manual for more information.

Note: If you want minimum hassle, we found that Red Hat 7.1 contains all dependencies for Aubit 4gl out of the box, on one CD, all in RPM automatic installs (It does NOT have PDFlib, since it is commercial product - OpenSource, but you will need to pay fee if you use it) It has unixODBC with GUI configuration tools, PostgreSQL with nice GUI tools, and all worked immediately for me. So if you have choice of operating systems, this is our recommended platform.

If you have RPM, or other package manager based system (DEB, etc), easiest way to check for existing dependencies is to query that system. Of course, if dependencies where installed using them...

ODBC

How to check if you have it:

UNIX: Safest bet is to look for file "odbc.ini" - all ODBC managers and/or drivers use it. If you have it, look for libodbc.so (unixODBC) or libiodbc.so (iODBC). 

You don't need ODBC manager, you can link with ODBC drivers directly, but this is probably not what you want (in this case, you will have to recompile everything to change database backend)

Windows: all Windows installation include ODBC manager, odbc32.dll

PLEASE NOTE: Whatever you do, verify that your ODBC connection is working, BEFORE you start screaming "Aubit is not working"! Thank you.

Download and install it if you don't: 

  • unixODBC (recommended)
  • OpenLink (Commercial product - uses custom version of iODBC manager)
  • iODBC (Standalone ODBC manager, GPL license)

Note that SOME managers come with drivers for SOME databases, and some don't. If you will be linking Aubit against ODBC manager, you will of course need both the manager, AND the ODBC driver(s) for your database(s) of choice. YMMW.

PDF

How to check if you have it:

UNIX: Look for file "pdflib.h" - Without it, you will be unable to compile Aubit with PDF support. If you downloaded precompiled Aubit binary with PDFlib support, look for "?????.so"

Windows: FIXME: Is PDFlib available for CygWin?

Download and install it if you don't: 

  • PDFlib

NOTE: This is commercial product - OpenSource, but you will need to pay fee if you use it. See license agreement on the site. 

NCURSES

How to check if you have it: at least version 5.0 or higher

UNIX: Look for "ncurses.h", probably in /usr/include. It should contain string NCURSES_VERSION_MAJOR

Windows: CygWin default installation includes ncurses

Download and install it if you don't:

  • Ncurses home page
  • Ncurses FTP site

GTK

How to check if you have it:

UNIX: Try "gtk-config --version"

Windows: try same as for UNIX, inside CygWin shell - see Aubit manual

Download and install it if you don't: 

  • GTK
  • GTK for Windows/CygWin

IksEmEl - Jabber and XML

(Needed only to compile PROGRAMS that will use XML/Jabber calls)

How to check if you have it:

UNIX: Look for file libiksemel.so

Download and install it if you don't: 

IksEmEl libraries

 

RPCgen

(Needed only when compiling Aubit 4gl compiler from source code)

How to check if you have it:

Windows/CygWin: Currently, you don't need RPCgen, since we are distributing Aubit code with special customized (read: hacked) versions of files that are on UNIX created by RPCgen. Reason for this is that most versions we found, where broken in one way or the other.

UNIX:

Download and install it if you don't: 

  • Sun RPCgen for CygWin readme
  • Sun RPC protocol specifications V2
  • Sun sources download
  • Netbula RPCgen (and online manual) - commercial
  • Please let me know if you have better link for RPC download
