Aubit 4gl automatic nightly builds download page

This page is here primarily for nightly automatic binary builds; it contains completely untested code, whatever is in CVS that night, will end up here. In short, stuff here is for people that like to live on bleeding edge.

Regular user's preferred option should be to download RELEASED files from Aubit project page, and/or to get source code from Aubit CVS - unless instructed otherwise, or if files available as released are seriously out of date.

Available for download:

  • Source distribution
  • Precompiled binary distribution
  • Other files
  • Optional functionality and dependencies download links

Please note: 

  • Files that don't have HTML links are not yet available
  • Files that have word "template" in there URL, failed to compile on last nightly build, and are not available today. Maybe tomorrow they will compile, and be available. We keep builds from previous days available, so you can click here to see if you may find what you want. Tip: look for full file names in your browser's status bar.

Source distribution

Note: Aubit 4gl is intensive development, and changes are made almost every day. Therefore, it is highly recommended that you get source code directly from CVS, to be able to update then using CVS client, instead downloading everything over and over again. See instruction on Aubit CVS page.

Note: TarBall, ZipBall and SRPM source code packages contain same code, packaged and compressed with different tools, unless they have different version or date in there name

Download source code (all platforms): 

  • Source TarBall in state used to compile UNIX binary distributions on this page
  • Source ZipBall in state used to compile Windows binary distributions on this page (Windows/CygWin) 
  • SRPM - Source code RPM package for use on platforms with RPM package tools
  • Go to CVS on SourceForge
  • Nightly CVS tree tarball - Includes EVERYTHING in CVS

Precompiled binary distribution

Platforms available:

  • Linux tar.gz and RPM precompiled binary packages
  • Microsoft Windows
  • Other operating systems

Please note:

  • All binary builds now include dynamically loadable support for libraries/functionality external to Aubit compiler, that you may or may not have on your system, like ODBC, PDF, GTK+, Jabber, etc. Packaged with binary distribution is a "configure" script, that will take care of available options detection, and initial configuration. If an option, like PDFlib library is not detected, this functionality will be disabled in automatically created configuration file. See "README.txt" file in archive you will download for details.
  • RPM package have a script that needs to be performed as part of RPM installation. If your RPM client is configured in a way which prohibits execution of scripts, installation will fail. If this happens, you will need to execute 'make' in Aubit RPM installation directory.
  • RPM installation will perform auto detection and create new 'aubitrc' file, but it will not install it if one already exists in /etc/opt/aubit4gl/aubitrc. You will see the message to this effect, if you are installing using command line RPM client, but not if you are using most GUI RPM clients. As a general rule, it is recommended to install using command line RPM client, since when using GUI clients, installation messages will not be visible to the user.
  • Typical size of tar.gz packages is between 2 and 5 MB
