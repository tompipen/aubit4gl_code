GET_INFO parameters

# indicates supply a number
eg. Name# Name1 = name for column 1
Name2 = name for column 2

"Window"

Info Returns:

"Height", Window Height (Integer)
"Width", Window Width (Integer)
"BeginX", Left position on screen of top left corner(Integer)
"BeginY", Y Position on screen of top left corner (Integer)
"Border", Does window have a border (true/false - Integer)
"Metrics", BeginX, BeginY,Width,Height
"CommentLine", Windows comment line(Integer)
"MenuLine", Windows menu line(Integer)
"PromptLine", Windows prompt line(Integer)
"FormLine", Windows form line(Integer)
"MessageLine", Windows message line(Integer)
"Color" Windows colour (Integer)

"Statement"

"NoColumns", Number of columns (Integer)
"NoRows", Number of Rows (Integer)
"Name#", Column Name for column # (char)
"Type#", Column type for column # (integer)
"Scale#", Column Scale for column # (integer)
"Nullable#", Can column # contain a null (true/false - integer)
"Length#" Column length for column # (integer)

"Form"

"Database", Database name on form
"Delimiters", Delimiters as specified on the form
"ScreenRecordCount", Number of screen records
"ScreenRecordName#", Screen Record name for record #
"FieldCount", Number of fields on the form
"FieldName#", Field name for field #
"Field#", Curses FIELD pointer for field #
"AttributesCount", Number of attributes
"CurrentField", Current field
"Width", Form Width
"Height" Form Height
