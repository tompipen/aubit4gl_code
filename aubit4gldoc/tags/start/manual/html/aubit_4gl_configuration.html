.a4glrc configuration file

This file is processed only by fglpc script (main 4gl file compiler and linker), in the following order:

  1. it is read from $AUBITDIR location, where it must exist
  2. it the file with this name is found in your $HOME, the settings found there will override the settings previously taken from $AUBITDIR

Please note that fglpc compiler is also used in the process of compiling the Aubit4gl compiler from source files distribution.

example .a4glrc file:

# Aubit 4gl **compiler** configuration file
#
#location of ODBC libraries (iODBC manager or other)
ODBC_LIB_DIR=/usr/openlink/lib

#where Aubit 4gl is installed: here you can override aubitenv if you need
#AUBITDIR=/usr/aubit/Aubit4gl

#optionally you can specify separate location for NCURSES
#NCURSES_LIB_DIR=/opt/home/fglcomp/ncurses/lib

#defines command executed as C compiler for your system
GCC="cc -O"

#To link a4gl with Informix CLI ODBC driver:
#ODBC_LIB_NAME=ifcli
#ODBC_LIB_DIR=$INFORMIXDIR/lib/cli

#Defines explicit libraries for use with non-iODBC manager drivers,
#like Informix CLI (now called just Informix ODBC) that is part
#of free Informix Client SDK (CSDK). If you will be using iODBC, do not
#use this, it is default
#ODBC_SHARED="-L$INFORMIXDIR/lib/cli -lcli -ldl"
#ODBC_STATIC="-L$INFORMIXDIR/lib/cli -lcli -ldl"
  • AUBITDIR
    Specifies the location of the directory the Aubit 4gl compiler is installed in 

    Default: NONE
  • NCURSES_LIB_DIR
    Optionally you can specify separate location for NCURSES libraries, for instance, when you have different versions of NCURSES libraries installed on the system.

    Default: like for all other libraries, the LD_LIBRARY_PATH (or other appropriate setting depending on your platform and compiler) is used
  • GCC
    Defines command to execute for C compiler

    Default: "gcc"
  • ODBC_LIB_DIR
    Specifies the location of ODBC libraries. Typically, this will be ODBC manager installation directory like iODBC manager or other manager, but it can also point to the locattion of the specific ODBC driver, like in the example file, for Informix ODBC driver direct linking (Informic CLI from Informix CSDK package)

    Defaults: LD_LIBRARY_PATH
  • ODBC_SHARED

    Default:
  • ODBC_STATIC

    Default:
  • ODBC_LIB_NAME

    Default: "iodbc' (The iODBC manager library)

General UNIX environment config related to Aubit 4GL

Typically, this environment variable will already be present in your startup/login scripts. We recommend that you use 'aubitenv' script to modify and/or add this kind of settings, and include a call from your system login scripts (typically /etc/profile) or user specific login scripts to this file.

 

  • LD_LIBRARY_PATH
    Must include at least $AUBITDIR/lib for A4GL compiler to work properly

  • PATH
    Must include at least $AUBITDIR/bin for A4G compiler to work properly

Runtime environment

This environment variables will typicaly be used at runtime, and therefore we recommend that you use 'aubitenv' script to modify and/or add this kind of settings, and include a call from your system login scripts (typically /etc/profile) or user specific login scripts to this file.

 

  • SQLUID
    Username to be used for database connection. Also used for DEFINE LIKE syntax support when compiling 4gl code

        Default: NONE
  • SQLPWD
    Password to be used for database connection. Also used for DEFINE LIKE syntax support when compiling 4gl code

        Default: NONE
  • HELPTEXT
    Sets the title for the help window

        Default: NONE
