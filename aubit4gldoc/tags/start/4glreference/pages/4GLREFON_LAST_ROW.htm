ON LAST ROW

The ON LAST ROW control block specifies the action that INFORMIX-4GL is to take after it processes the last input record that was passed to the REPORT routine and encounters the FINISH REPORT statement.

statement is a Report execution statement or other 4GL statement.

The statements in the ON LAST ROW control block are executed after the statements in the ON EVERY ROW and AFTER GROUP OF control blocks.

When INFORMIX-4GL processes the statements in an ON LAST ROW control block, the variables that the report is processing still have the values from the final record that the report processed. The ON LAST ROW control block can use aggregate functions to display report totals.
