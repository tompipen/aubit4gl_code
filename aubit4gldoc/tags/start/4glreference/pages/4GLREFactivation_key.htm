activation key

A logical key that the developer defines to provide the user with some programmer-defined feature. The developer can define an activation key in the ON KEY clause of the CONSTRUCT, DISPLAY ARRAY, INPUT, INPUT ARRAY, or PROMPT statement; or in the KEY clause of the MENU statement. When the user presses the activation key, 4GL executes the control block associated with that key.

See also control block, key, logical key, menu option.
