printable character

A character that can be displayed on a terminal or printer. These characters include the ASCII codes 32 through 126: A-Z, a-z, 0-9, symbols (!, #, $, ;, *, and so on), TAB (CONTROL-I), NEWLINE (CONTROL-J), FORMFEED (CONTROL-L) and the blank space character ( ).

See also ASCII, blank space, character.
