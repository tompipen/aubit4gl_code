report

A 4GL program block defined with the REPORT statement. A report formats data, sent as input records. The report header follows the REPORT keyword and defines the name and formal argument list (the input record) for the report. The report body (all statements between the report header and the END REPORT keywords) defines the actions of the report.

See also argument, control block, function, input record, page header, page trailer, programmer-defined function, output file, program block.
