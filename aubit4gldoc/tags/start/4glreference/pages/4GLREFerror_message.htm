error message

Text that describes a 4GL error. Each error is identified by an integer, usually negative, called an error code. Each code corresponds to a specific error message. Such messages can be retrieved by running the Error Reference utility or within a program by making a call to the ERR_GET( ), ERR_PRINT( ), or ERR_QUIT( ) built-in functions. By default, 4GL automatically displays some run-time error messages on the screen.

See also error handling, error log, status variable.
