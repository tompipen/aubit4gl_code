relational operators

Operators that perform comparison operations. These operators return the values TRUE (=1), FALSE (=0), and in some cases UNKNOWN (If an operand evaluates to NULL, Boolean operators can yield a third unknown result that 4GL treats as FALSE). 4GL relational operators are: equal (=), not equal (!= or <>), greater than (>), less than (<), greater than or equal to (>=), less than or equal to (<=). All relational operators have the same precedence level.

See also associativity, binary operator, Boolean operator, operator, precedence.
