calling routine

The program block that calls a function (or report). In 4GL, the calling routine can be either a MAIN, a FUNCTION, or a REPORT program block. This call can be explicit, by means of the CALL statement, or implicit, by embedding the function name within an expression. The calling routine can pass in values through the function arguments and can receive return values (if they are defined within the function).

See also argument, expression, function call, program block, return value.
