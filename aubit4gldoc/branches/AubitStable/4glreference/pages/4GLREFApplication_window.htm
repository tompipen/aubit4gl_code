Application window

  1. In the INFORMIX-4GL Interactive Debugger, the Application window is where the Debugger displays the 4GL application.

  2. When running a 4GL application, the Application window is the separate GUI window that contains the display area (with the applications forms, 4GL windows, and text) as well as the mouse bar. This window provides a user interface using the native GUI format. The Application window always contains at least one 4GL window, the screen.

See also 4GL window, display area, Help window, Line-mode overlay, mouse bar, Report window, screen.
