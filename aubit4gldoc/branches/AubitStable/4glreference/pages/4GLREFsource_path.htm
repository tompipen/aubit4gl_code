source path

A list of directories in which the INFORMIX-4GL Interactive Debugger searches for the source modules of the 4GL application that you are debugging. The source path is analogous to the search path stored in the PATH environment variable. You can set the source path with the DBPATH or DBSRC environment variables, with a command line option, or with the USE command.
