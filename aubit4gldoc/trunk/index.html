Aubit 4GL project documentation menu

Available on-line documents:

  • x-4GL Language Reference Manual (CVS)
  • Aubit 4GL compiler Users Manual (CVS)
  • lib4gl (Power-4GL) Users Manual (CVS)
  • fglgen - 4GL code generators Users Manual
  • fgldoc - 4GL code documentation tool Users Manual
  • Hi4GL compiler User's manuals menu (CVS)
  • Errata and additional on-line documents (on-line only)
  • For Aubit 4GL compiler developers: Aubit 4GL compiler source code internals documentation, on-line only (DoxyGen)

Some of this manuals are available for download as compressed archives from Aubit 4GL project page, and all are available from Aubit CVS. Hy4GL manuals are included in Hy4GL distribution/CVS, and all other manuals are in CVS under "aubit4gldoc" tree.

See Aubit CVS main page for instruction on downloading from CVS.

Please report documentation errors to Andrej Falout or to Aubit 4gl mailing list.
