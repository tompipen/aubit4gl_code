Aubit 4GL Manual

Overview

This fourth generation language compiler is largely based on Informix 4GL, but using the power of ODBC for full connectivity to any ODBC data source. It has been designed for programmers,  with a host of features enabling programs to be written which were just not possible before, while retaining compatibility with existing I-4gl source code. 

Source code for compiler is released under GPL, and for libraries under LGPL license.

Aubit 4gl development home is at SourceForge .You will always be able to find up-to-date version of this manual there, on-line.

Initial code was written by Mike Aubury of www.aubit.com and you can purchase commercial support from them.
