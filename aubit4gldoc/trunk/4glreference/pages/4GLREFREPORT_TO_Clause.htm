REPORT TO Clause

This clause specifies a default destination for output from the report. This destination can be a file, an operating system pipe, or the system printer.

command is a quoted string, specifying a program, shell script, or command line to receive the output from the report.

filename is a quoted string, containing the name of a file to receive the report output. This filename can also include a path name.

  • ariable is the name of a CHAR or VARCHAR variable that contains a shell script or command line to receive the output from the report.

If a valid START REPORT statement includes a TO clause directing output of the report to some destination, that START REPORT statement is executed, and any REPORT TO clause in the OUTPUT section has no effect.
