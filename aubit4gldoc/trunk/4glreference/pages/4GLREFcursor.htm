cursor

  1. A focal point at which action can be applied. A text cursor is a mark showing the focal point for keyboard input. A mouse cursor shows the point at which a mouse click will take effect.

See also keyboard focus, mouse, text cursor.

  1. A database cursor is an identifier associated with an active set. It points to the current row in the active set. This row can be fetched, deleted, or updated. 4GL supports the following types of database cursors: sequential, scroll, hold, update, and insert.

See also active set, close, current, identifier, open, prepared statement, query, row, scrolling.
