SLEEP

The SLEEP statement suspends execution of the 4GL program for a specified positive whole number of seconds. It can also respond to GUI screen refresh events that accumulated in the event queue since the last interactive statement.

The SLEEP statement causes the program to pause for the specified number of seconds. This can be useful, for example, when you want the screen display to remain visible long enough for the user to read it.
