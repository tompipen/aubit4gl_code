FGL_SETENV()

A-4gl V 0.2

The FGL_SETENV() receives a character string, that you want exported to current running program environment, including variable name and variable value, separated by equal sign ("=").

 

    FGL_SETENV __________ ( "[character expression]" ) ___________| 

 

FGL_SETENV() function returns TRUE on success, and FALSE on failure to perform it's task.

Example:

    call fgl_setenv("DBPATH=/usr/app/myforms")
