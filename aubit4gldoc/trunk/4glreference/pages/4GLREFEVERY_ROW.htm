EVERY ROW

The EVERY ROW keywords cause INFORMIX-4GL to include in the output every input record that is passed to the report. If you use the EVERY ROW option, you cannot specify any other statements or control blocks.

This option formats the report in a simple default format, containing only the values that are passed to the REPORT routine through its arguments, and the names of the arguments.

You cannot modify the EVERY ROW statement with any of the statements listed under Statements Valid Only in the FORMAT Section, nor can you include any control blocks in the FORMAT section. To display every record in a format other than the default format, use the ON EVERY ROW control block.
