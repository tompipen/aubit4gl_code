logical key

A key (or button) that the user can press to perform certain tasks predefined by 4GL or the operating system. These keys include the following: Accept key, Delete key, Help key, Insert key, Interrupt key, Quit key. Each key is associated with some default physical key. The OPTIONS statement can assign most logical keys to different physical keys. Certain 4GL statements can reference logical keys with keywords like ACCEPT, DELETE, INSERT, and so forth.

See also Accept key, Delete key, Help key, Insert key, Interrupt key, key, mouse bar, Quit key.
