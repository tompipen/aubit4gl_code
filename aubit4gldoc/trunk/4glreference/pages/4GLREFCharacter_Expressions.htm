Character Expressions

A character expression is a specification that evaluates to a character string.

character is one or more characters enclosed between two single ( ' ) or double ( " ) quotation marks. (This is sometimes called a character string, a quoted string, or a string literal.)

format-string is a quoted string of symbols to specify how 4GL displays the returned character value.

Here the function call or named value returns a CHAR or VARCHAR value.

No variable in a character expression can be of the TEXT data type, except in a NULL test, or as a WORDWRAP operand in a PRINT statement of a 4GL report. As in any 4GL statement or expression, you cannot reference a named value outside its scope of reference.
